# GameStore

## 배포

#### AWS를 활용한 배포
1. IAM을 통한 권한과 역할 생성
2. EC2 가상 서버로 애플리케이션 실행
3. RDS를 활용한 데이터베이스 설정 수정
4. Elastic Beanstalk를 이용한 배포 자동화
